Описание предметной области

Сайт является путеводителем по миру аниме. Любой пользователь может получить информацию об аниме: название, жанр, кол-во серий, рейтинг, дата выхода, статус выхода, автора, студию, персонажей в аниме. Помимо этого, можем узнать об авторе, его ФИ, дату рождения и аниме, которые он создал. Также, мы можем узнать студию(название), рисовавшее аниме(реализовывали идеи автора). У каждого аниме есть персонажи, которые имеются в аниме. Один и тот же персонаж может быть в разных аниме. У них есть имя и фамилия, а также уникальный псевдоним. Рейтинг устанавливает администрация. У каждого аниме имеется несколько жанров. Функционал похож на википедию по миру аниме.
